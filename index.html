FAQ

Fair questions.

Does it work in multiplayer?

Yes — that's the point. The crew sits on the SRS frequencies like human controllers; everyone on the server can talk to it. Only the host runs WILCO, friends just tune the radio.

Is it cheating?

No. It's a controller, not a sensor: it tells you what a real controller with a real scope would say. It never gives you information a human GCI couldn't.

What does it cost to run?

Nothing during the beta — your invite code covers the AI. There's a fair daily allowance (enough for a long evening of flying) and it resets every day. Voices are included.

Will it break server integrity checks?

No core files are touched. The data feed is a single hook file in Saved Games — the same mechanism SRS and LotATC use.

What do I need?

Windows 10/11, DCS World, SRS, and a mid-range PC. A GPU makes speech recognition faster, but CPU works. Setup is one script.

Can it handle my accent / sloppy radio?

It's built for real pilots, not voice-command grammar. It handles hesitation, slang, accents and half-finished calls — and asks you to "say again" like a human would when you genuinely mumble.
